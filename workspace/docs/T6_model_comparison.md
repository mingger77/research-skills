# 模型对比分析：集总参数 vs 连续体（T6）

## 一、两种模型概述

| 维度 | 集总参数模型（2-DOF） | 连续体波动模型（FTCS） |
|------|----------------------|----------------------|
| **物理图景** | 双球作为质点，橡皮筋简化为弹簧 | 橡皮筋作为一维连续弹性介质 |
| **数学形式** | 二阶 ODE（常微分方程） | 一维波动 PDE（偏微分方程） |
| **求解方法** | scipy.integrate.solve_ivp (RK45) | 有限差分 FTCS 显式格式 |
| **计算量** | 小（秒级） | 较大（需满足 CFL 条件） |
| **适用场景** | 系统级周期分析、参数扫描 | 波动过程、驻波模式、谐波分析 |

## 二、关键参数对应关系

### 2-DOF 模型参数（来自 PLAN.md）

| 参数 | 值 | 来源 |
|------|-----|------|
| 小球质量 m | 0.01 kg | 实验器材 |
| 公转半径 R | 0.03 m | 实验装置 |
| 转动惯量 I = mR² | 9.0×10⁻⁶ kg·m² | 计算值 |
| 扭转刚度 k | 0.05 N·m/rad | 文献估算 |
| 阻尼系数 c | 5×10⁻⁵ N·m·s/rad | 弱阻尼假设 |
| **扭振周期 T** | **0.0596 s（16.8 Hz）** | 理论/数值一致 |

### 连续体模型参数（来自 PLAN_v2.md）

| 参数 | 值 | 来源 |
|------|-----|------|
| 橡皮筋长度 L | 0.12 m | 实验器材 |
| 剪切模量 G | 0.4 MPa | 橡胶典型值 |
| 密度 ρ | 930 kg/m³ | 橡胶典型值 |
| 波速 v = √(G/ρ) | 20.74 m/s | 计算值 |
| **基频 f₁ = v/(2L)** | **86.4 Hz** | 连续体理论 |

## 三、模型对比

### 频率差异

集总参数模型给出的扭振频率为 ~17 Hz，连续体模型的基频为 ~86 Hz。两者相差约 5 倍。原因分析：

1. **橡皮筋几何效应：** 集总参数模型的扭转刚度 k 包含了橡皮筋长度、截面形状的积分效应。连续体模型的 G 是材料剪切模量，两者之间的换算关系为 k = GJ/L（J为截面极惯性矩）。对一根直径 d=1mm 的橡皮筋，J = πd⁴/32 ≈ 9.8×10⁻¹⁴ m⁴，则 k = GJ/L ≈ 4×10⁵ × 9.8×10⁻¹⁴ / 0.12 ≈ 3.3×10⁻⁷ N·m/rad。远小于估算值 0.05。

2. **结论：** 两个模型的差异主要源于橡皮筋的**等效扭转刚度**参数取值。集总参数模型中，k=0.05 包含了公转半径 R 的放大效应（杠杆效应），而连续体模型直接使用材料剪切模量。

### 互补性

| 现象 | 集总参数模型 | 连续体模型 |
|------|:-----------:|:---------:|
| 周期/频率预测 | ✅ 系统级准确 | ⚠️ 需准确几何参数 |
| 参数扫描趋势 | ✅ T ∝ √(m)/√(k) | ✅ f ∝ √(G)/L |
| 阻尼效应 | ✅ 指数衰减模拟 | ❌ 未实现 |
| 非线性 | ✅ Duffing 非等时性 | ❌ 未实现 |
| 驻波模式 | ❌ 无法描述 | ✅ 节点/模式精准 |
| 波传播过程 | ❌ 无法描述 | ✅ 动画直观可见 |
| 谐波分析 | ❌ 仅基频 | ✅ 多阶频率响应 |

## 四、综合结论

- **集总参数模型（2-DOF）** 适合系统级的周期预测和参数依赖性分析。计算快、趋势准确，适合与实验数据做定量对比。
- **连续体波动模型** 适合理解橡皮筋上的波动传播机制、驻波模式的形成、共振频率等空间分布特性。节点位置误差 < 3%（Mode 2/4），理论预测准确。
- **建议：** 后续研究可将两种模型结合使用：
  1. 用 2-DOF 模型快速拟合实验数据，反推等效参数
  2. 用连续体模型验证驻波模式，理解波动本质
  3. 根据实测的橡皮筋几何尺寸，修正两者的参数对应关系
