## 一、目标拆解

### 子任务 T1：二自由度扭振模型数值求解

| 项目 | 内容 |
|------|------|
| **输入** | 物理参数：小球质量 m₁=m₂、公转半径 R、扭转刚度 k、阻尼系数 c、初始扭转角 θ₀、初始角速度 ω₀=0 |
| **输出** | 扭振 ODE 求解结果：角度 θ₁(t)、θ₂(t) 及角速度 ω₁(t)、ω₂(t) 随时间变化 |
| **成功标准** | 1. solve_ivp 正常收敛，无数值发散<br>2. 无阻尼时系统总能量守恒<br>3. 有阻尼时振幅呈指数衰减 |
| **失败判定** | 1. 数值解发散（NaN/Inf）<br>2. 能量不守恒（无阻尼情形） |

**物理模型：**

二自由度线性阻尼扭振系统：

```
状态向量: x = [θ₁, θ₂, ω₁, ω₂]ᵀ

dθ₁/dt = ω₁
dθ₂/dt = ω₂
dω₁/dt = [-k(θ₁-θ₂) - c(ω₁-ω₂)] / I
dω₂/dt = [ k(θ₁-θ₂) + c(ω₁-ω₂)] / I

其中 I = m·R² 为小球转动惯量
```

**数值方法：** `scipy.integrate.solve_ivp`，method='RK45'，rtol=1e-8

### 子任务 T3：周期非等时性分析

| 项目 | 内容 |
|------|------|
| **输入** | T1 求解器 + 不同初始扭转角 θ₀ |
| **输出** | 周期-初始振幅关系曲线、相空间轨迹图 |
| **成功标准** | 1. 线性阻尼模型下能观察到周期随振幅变化<br>2. 若线性模型不足以产生非等时性，记录并进入备选方案 |
| **失败判定** | 线性模型下周期完全不变（确认等时性），需引入非线性 |

**分析方法：**
1. 从 θ̈ + 2βθ̇ + ω₀²θ = 0 的解析周期公式判断线性模型的等时性
2. 若线性模型周期与振幅无关 → 引入非线性恢复力矩项（k₂θ³）
3. 比较线性和非线性模型的相空间轨迹

**非线性扩展（备选）：**
- 非线性恢复力矩: M = -k₁θ - k₃θ³
- 运动方程: Iθ̈ + cθ̇ + k₁θ + k₃θ³ = 0
- 此时周期 T 随振幅增大而变化（硬弹簧/软弹簧行为）

### 核心伪代码

**model.py：**
```python
def torsional_ode(t, x, I, k, c, nonlinear=False, k3=0):
    """二自由度扭振系统 ODE 右端函数"""
    θ₁, θ₂, ω₁, ω₂ = x
    Δθ = θ₁ - θ₂
    Δω = ω₁ - ω₂
    # 恢复力矩（可选非线性项）
    if nonlinear:
        T_restore = k*Δθ + k3*Δθ**3
    else:
        T_restore = k*Δθ
    # 阻尼力矩
    T_damp = c * Δω
    # 角加速度
    α₁ = -(T_restore + T_damp) / I
    α₂ =  (T_restore + T_damp) / I
    return [ω₁, ω₂, α₁, α₂]
```

**solver.py：**
```python
def solve_torsion(I, k, c, θ₀, t_span, dt=0.01, **kwargs):
    """包装 solve_ivp，返回角度/角速度时间序列"""
    x₀ = [θ₀, -θ₀, 0, 0]  # 初始条件：两球反向扭转
    t_eval = np.arange(0, t_span, dt)
    sol = solve_ivp(torsional_ode, [0, t_span], x₀,
                    args=(I, k, c), t_eval=t_eval,
                    method='RK45', rtol=1e-8)
    return sol.t, sol.y
```

**scan.py：**
```python
def scan_parameter(param_name, param_values, fixed_params, t_span):
    """扫描单一参数，计算每个参数值下的周期"""
    periods = []
    for val in param_values:
        params = {**fixed_params, param_name: val}
        t, y = solve_torsion(**params, t_span)
        period = extract_period(t, y[0])  # 从 θ₁ 提取周期
        periods.append(period)
    return param_values, periods
```

## 四、可能困难及备选方案

| 困难 | 备选方案 |
|------|----------|
| **线性模型无法产生周期非等时性**（简谐振动的周期本就与振幅无关） | 在恢复力矩中引入非线性项 k₃θ³，使用 Duffing 振子模型 |
| **缺少实验实测参数**（k、c 不知道具体值） | 采用文献参考值，并通过参数扫描覆盖合理范围；写配置方便后续替换 |
| **二自由度模型不能完全解释驻波现象** | 扩展为连续体模型，用有限差分法求解一维波动方程，观察驻波模式 |
| **长期数值积分能量衰减/发散** | 检查能量守恒，必要时改用 symplectic integrator 或减小 tolerances |

---

## 五、Baseline 方案

1. 使用 `scipy.integrate.solve_ivp(RK45)` 求解二自由度 **线性** 扭振 ODE
2. 固定参数运行一次，输出 θ(t) 和 ω(t) 时域曲线
3. 计算振动的周期（过零点检测法）
4. 与理论周期公式 T = 2π√(I/(2k)) 对比验证
5. 确认数值解正确后，进行参数扫描

**Baseline 验收标准：** 数值周期与理论周期误差 < 1%
