## 一、动机：为什么需要连续体模型

第一轮（PLAN.md）实现了二自由度**集总参数**扭振模型，能够：
- 高精度求解 ODE（周期误差 0.002%）
- 参数扫描和趋势分析
- 线性/非线性对比（周期非等时性）

**但局限：** 集总参数模型将橡皮筋简化为弹簧，无法描述橡皮筋上的波动传播过程，不能直接模拟"驻波"现象。

**本轮目标：** 将橡皮筋建模为一维连续弹性介质，用有限差分法求解扭转波动方程，观察驻波模式的形成和演化。

## 二、目标拆解

### 子任务 T4：一维扭转波动方程数值求解

| 项目 | 内容 |
|------|------|
| **输入** | 橡皮筋长度 L、剪切模量 G、密度 ρ、激励频率 f、边界条件类型 |
| **输出** | 波函数 u(x,t) 时空分布、动画 GIF |
| **成功标准** | 1. 波动能量守恒（无阻尼情形）<br>2. 驻波节点位置与理论值一致（λ=2L/n） |
| **失败判定** | 数值发散（CFL条件不满足）、波速与理论不符 |

**物理模型：**

一维扭转波动方程：

```
∂²u/∂t² = v² · ∂²u/∂x²    其中 v = √(G/ρ)
```

其中 u(x,t) 为橡皮筋上位置 x 在时间 t 的扭转角位移。

**边界条件（两种）：**
- 两端固定（驻波）：u(0,t) = u(L,t) = 0
- 一端固定、一端激励：u(0,t) = A·sin(ωt), u(L,t) = 0

**数值方法：** 中心差分显式格式（FTCS）：

```
u(x, t+Δt) = 2u(x,t) - u(x, t-Δt) + (v·Δt/Δx)² · [u(x+Δx,t) - 2u(x,t) + u(x-Δx,t)]
```

**CFL 条件：** v·Δt/Δx ≤ 1

### 子任务 T5：驻波模式分析

| 项目 | 内容 |
|------|------|
| **输入** | T4 的波动求解结果 |
| **输出** | 各阶驻波模式图、节点位置表、频率-阶数关系图 |
| **成功标准** | 驻波节点位置与理论 λ_n = 2L/n 一致，误差 < 5% |
| **失败判定** | 节点位置偏差 > 10% |

**理论预期：**
- 基频 f₁ = v/(2L)
- 第 n 阶频率 f_n = n·v/(2L)
- 第 n 阶节点位置 x_j = j·L/n, j=0,1,...,n

| 项目 | 内容 |
|------|------|
| **输入** | T1 的 2-DOF 结果 + T4 的连续体结果 |
| **输出** | 对比分析表、讨论文档 |

**对比方法：** 
- 连续体模型的基频应与 2-DOF 模型的扭振频率可比较

### 核心伪代码（wave_model.py）

```python
def solve_wave_equation(L, G, rho, nx, nt, bc_type='fixed', excitation=None):
    """
    有限差分法求解一维波动方程。

    参数:
        L: 橡皮筋长度 [m]
        G: 剪切模量 [Pa]
        rho: 密度 [kg/m³]
        nx: 空间网格点数
        nt: 时间步数
        bc_type: 'fixed' 两端固定, 'driven' 一端激励
        excitation: 激励函数（当 bc_type='driven' 时使用）

    返回:
        u: 波函数矩阵 [nt, nx]
        x: 空间坐标
        t: 时间序列
    """
    dx = L / (nx - 1)
    v = sqrt(G / rho)
    dt = dx / v * 0.9  # CFL = 0.9

    u = zeros((nt, nx))
    u_prev = zeros(nx)

    # 初始条件：高斯波包
    u[0, :] = exp(-((x - L/2) / sigma)**2)

    # 时间推进
    for n in range(1, nt-1):
        for i in range(1, nx-1):
            u[n+1, i] = (2*u[n,i] - u_prev[i]
                         + CFL² * (u[n,i+1] - 2*u[n,i] + u[n,i-1]))
        # 应用边界条件
        u[n+1, 0] = 0           # 固定端
        u[n+1, -1] = 0          # 固定端
        # 或驱动端: u[n+1, 0] = A*sin(ω*n*dt)

    return u, x, t
```

## 五、可能困难及备选方案

| 困难 | 备选方案 |
|------|----------|
| **CFL 条件导致时间步过小，计算量过大** | 降低空间分辨率 nx，或使用隐式格式（无条件稳定） |
| **驻波节点定位不准确** | 对波函数进行插值后找零点，或使用时间平均法 |
| **连续体与集总参数模型频率不匹配** | 调整橡皮筋的有效剪切模量 G_eff，使其基频与 2-DOF 模型的扭振频率对应 |

## 七、验证方式

1. **能量守恒：** 无阻尼波动方程数值解的总能量应守恒
2. **CFL 稳定性：** v·Δt/Δx < 1 时稳定，> 1 时发散
3. **驻波节点：** 第 n 阶驻波有 n+1 个节点（含端点），位置与理论一致
4. **频率对应：** 橡皮筋基频 f₁ = v/(2L) 应与 2-DOF 模型的扭振频率可比
