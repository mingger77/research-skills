# log-write-constraint

这个 skill 旨在给以 Claude Code 为代表的 Agent 提供一套进行科研日志写作的方法约束。

## 日志结构

```
log/
├── EXECUTION_LOG.md
├── ENVIRONMENT_LOG.md
├── STAGE_LOG.md
├── SEARCH_LOG.md
├── EXPERIMENT_LOG.md
└── DATA_LOG.md
```

---

## 日志组织规范

### 1. EXECUTION_LOG.md

```
# 执行日志

- **[ACTION-[serial]]**
  - 操作类型：...
  - 时间戳：...
  - 输入：...
  - 输出：...
  - 涉及文件：...
  - 是否成功：...
  - 操作摘要：...
  - 备注：...
- **[...]**
  - ...
```

### 2. ENVIRONMENT_LOG.md

```
# 环境记录

- **[TIME-[serial]]**
  - 操作系统：...
  - CPU：...
  - GPU：...
  - CUDA：...
  - 依赖版本：...
  - 备注：...
- **[...]**
  - ...
```

### 3. STAGE_LOG.md

```
# 状态记录

- project_name: ...
- research_direction: ...
- research_goal: ...
- created_time: ...
- created_location: ...
- **[STAGE-[serial]]**
  - current_state: ...
  - current_plan: ...
  - active_plan: ...
  - loaded-skill: ...
```

### 4. SEARCH_LOG.md

```
# 资料搜集日志

- **[INFORMATION-[level]-[serial]]**（来源：[URL]）
  - 标题：...
  - 来源：...
  - 作者：...
  - 年份：...
  - 会议/期刊：...
  - DOI：...
  - 引用量：...
  - 检索时间：...
- **[...]**
  - ...

- 使用关键词：...
- 使用搜索引擎/数据库：...
- 筛选标准：...
- 最终保留的文献数量：...
```

### 5. EXPERIMENT_LOG.md

```
# 实验日志

- **[EXPERIMENT-[serial]]**
  - 事件：...
  - Git Commit：...
  - 数据集版本：...
  - 是否运行 Python 程序：[yes/no]
    - 参数：...
    - 随机种子：...
    - Python 版本：...
    - 依赖版本：...
- **[...]**
  - ...
```

### 6. DATA_LOG.md

```
# 数据处理日志

- **[DATA-[serial]]**
  - 处理时间：...
  - 原始数据文件：...
  - 输出文件：...
  - 处理操作摘要：...
  - 涉及的代码：...
  - 参数设置：...
  - 数据行数变化：X → Y
  - 备注：...
- **[...]**
  - ...
```
